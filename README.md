# 🎵 CoverFetch

Una herramienta web minimalista para obtener portadas de Spotify en la máxima calidad disponible.

## Características

- Soporte para Tracks
- Soporte para Álbumes
- Soporte para Playlists
- Diseño responsive
- Interfaz rápida y ligera

## Tecnologías

- HTML
- CSS
- JavaScript
